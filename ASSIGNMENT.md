## Project: Premier Schools Exhibition (PSE) — Landing Page

### Stack & Structure
- Semantic **HTML5 + custom CSS** — **no frameworks**.
- Follow **BEM** naming convention.
- Pass **W3C validation** (HTML/CSS).

### Accessibility
- **WCAG 2.2 AA** compliant.
- ARIA roles for sliders, skip-to-content link.
- Full keyboard + screen reader support.

### Responsiveness
- Fully responsive across desktop, tablet, and mobile.

### Cross-browser
- Chrome, Firefox, Safari, Edge (latest 2 versions) + iOS/Android browsers.

## Sections to build

1. **Hero Section** — Dual-axis slider (horizontal & vertical), auto-play, swipe,
   pause on hover, accessible controls.
2. **Participating School Logos** — Continuous "sling" marquee animation with
   alternating left→right and right→left flow; pause on hover/focus.
3. **Choose the School** — Four cards on desktop; convert to mobile slider with
   swipe + pagination dots.
4. **Exhibition Section** — Entire section as a slider with 3–6 highlight cards;
   accessible, consistent card height, auto-play optional.

## QA Criteria
- Validate HTML/CSS (W3C).
- Accessibility audit via **axe**.
- Test on key devices.
- Animations honor **prefers-reduced-motion**.
